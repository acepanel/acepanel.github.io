# 更新

## 自動更新

面板預設已啟用自動更新，每天凌晨 2 點左右檢查並更新。 更新期間面板會短暫無法使用（通常少於 1 分鐘）。

若要關閉自動更新：「設定」->「安全」->「自動更新」。

## 手動更新

### 網頁介面

點選首頁右上角的「更新」按鈕。 當有新版本可用時，會彈出更新頁面。

更新過程中請勿重新整理瀏覽器或操作面板。 完成後頁面會自動重新整理。

### 命令列

```shell
acepanel update
```

適用於面板無法存取的情況。 更新過程中請勿關閉終端機。

## 更新失敗

如果更新後出現問題，請嘗試修復：

```shell
acepanel fix
```

如果問題仍然存在，請至 [社群](https://tom.moe/c/technical/acepanel) 提供回饋。
