# 更新

## 自動更新

面板預設開啟自動更新，每天凌晨 2 點左右會自動檢測並更新。 更新期間面板會短暫不可用（通常不超過 1 分鐘）。

關閉自動更新：「設定」->「安全」->「自動更新」。

## 手動更新

### Web 介面

點擊首頁右上角「更新」按鈕， 有新版本時會彈出更新頁面。

更新過程中請勿重新整理瀏覽器或操作面板， 完成後頁面會自動重新整理。

### 命令列

```shell
acepanel update
```

適用於面板無法存取的情況。 更新過程中請勿關閉終端。

## 更新失敗

如果更新後出現問題，嘗試修復：

```shell
acepanel fix
```

仍有問題可到 [社群](https://tom.moe) 回饋。
