# 用戶管理

用戶管理頁面用於管理面板的登錄用戶。

![用戶管理](/images/setting/setting-user.png)

## 用戶列表

列表顯示所有面板用戶的信息：

| 字段   | 說明              |
| ---- | --------------- |
| 用戶名  | 登錄用戶名，可直接在表格中修改 |
| 郵箱   | 用戶郵箱，用於找回密碼等功能  |
| 兩步驗證 | 是否啟用 TOTP 兩步驗證  |
| 創建時間 | 用戶創建時間          |

## 創建用戶

點擊 **創建用戶** 按鈕添加新用戶：

![創建用戶](/images/setting/setting-user-create.png)

填寫以下信息：

- **用戶名**：登錄用戶名
- **密碼**：登錄密碼
- **郵箱**：用戶郵箱地址

## 用戶操作

### 訪問令牌

生成 API 訪問令牌，用於通過 API 接口訪問面板功能。

令牌具有與用戶相同的權限，請妥善保管。

### 更改密碼

修改用戶的登錄密碼。 建議定期更換密碼。

### 刪除用戶

刪除用戶賬號。 刪除後該用戶將無法登錄面板。

:::warning 注意
至少需要保留一個用戶，無法刪除最後一個用戶。
:::

## 兩步驗證

兩步驗證（2FA）為賬號提供額外的安全保護。 啟用後登錄時除了密碼還需要輸入動態驗證碼。

### 啟用兩步驗證

1. 點擊兩步驗證開關
2. 使用 Google Authenticator、Microsoft Authenticator 等應用掃描二維碼
3. 輸入應用顯示的 6 位驗證碼確認

### 支持的驗證器應用

- Google Authenticator
- Microsoft Authenticator
- Authy
- 1Password
- 其他支持 TOTP 的應用

## 安全建議

1. 使用強密碼，包含大小寫字母、數字和特殊字符
2. 啟用兩步驗證
3. 定期更換密碼
4. 不要共享賬號
5. 及時刪除不再使用的賬號
