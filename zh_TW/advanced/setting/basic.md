# 基本設定

基本設定頁面用於設定面板的基礎參數。

![基本設定](/images/setting/setting-basic.png)

## 面板名稱

自訂面板顯示的名稱，預設為「AcePanel」。 修改後重新整理頁面，即可在瀏覽器標題與面板左上角看到顯示。

## 語言

選擇面板介面語言：

- 簡體中文
- 繁體中文
- English

## 更新頻道

選擇面板更新的版本頻道：

- **穩定版**：經過完整測試的正式版本，建議在生產環境中使用
- **測試版**：包含最新功能但可能存在問題，適合搶先體驗的使用者

## 連接埠

面板的存取連接埠，預設為 8888。 變更連接埠後需要：

1. 確保新的連接埠未被佔用
2. 在伺服器安全性群組中開放新的連接埠
3. 儲存後面板會自動重新啟動

:::warning 注意
變更連接埠前，請確保伺服器安全性群組已開放新的連接埠，否則可能無法存取面板。
:::

## 預設目錄

設定各類資源的預設儲存路徑：

| 設定項目   | 預設值               | 說明        |
| ------ | ----------------- | --------- |
| 預設網站目錄 | /opt/ace/sites    | 存放網站檔案的目錄 |
| 預設備份目錄 | /opt/ace/backup   | 存放備份檔案的目錄 |
| 預設專案目錄 | /opt/ace/projects | 存放專案檔案的目錄 |

變更目錄後，新建立的網站／備份／專案會使用新目錄，現有的不受影響。

## 備份壓縮格式

選擇建立備份時使用的壓縮格式。 可選項目有 `tar.xz`（預設）、`tar.gz`、`tar.zst`、`zip` 和 `7z`。 選擇 `tar.zst` 可在速度與壓縮率之間取得良好平衡，或選擇 `zip`/`7z` 以獲得更好的跨平台相容性。

## 容器 Socket

指定容器執行環境 Socket 的路徑，預設為 `/var/run/docker.sock`。 當使用非預設的 Docker Socket 位置或 Podman Socket 時，請調整此項目，以便容器管理模組能正確連線。

## IP 資料庫

設定用於解析訪客 IP 地理位置的 GeoIP 資料庫（例如用於網站統計與防火牆掃描感知模組）。 提供三種模式：

- **停用**：不解析 IP 地理位置。
- **訂閱**：提供訂閱網址（預設 `https://fastly.jsdelivr.net/npm/qqwry.ipdb/qqwry.ipdb`）。 資料庫每週自動更新，且必須為 IPIP.NET 格式（`.ipdb`）。
- **自訂檔案**：提供本機 IPIP.NET 格式 `.ipdb` 資料庫檔案的路徑。

## 自訂 Logo

輸入圖片的完整 URL 網址，即可替換面板左上角的 Logo。 支援 PNG、JPG、SVG 等格式。

## 隱藏選單

選擇要隱藏的選單項目。 隱藏後的選單不會顯示在左側導覽列中，但功能仍可使用（透過直接存取 URL）。

適用於簡化介面，或限制一般使用者可見的功能範圍。
