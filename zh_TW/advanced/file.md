# 檔案

檔案模組提供了功能強大的圖形化檔案管理器， 設計理念是盡可能還原 Windows 資源管理器的操作體驗，支援右鍵選單、拖曳上傳、快捷鍵等特性。

## 檔案管理器

![檔案管理器](/images/file/file-list.png)

## 核心特性

### 類 Windows 操作體驗

- **右鍵選單**：右鍵點擊檔案或目錄彈出操作選單
- **拖曳上傳**：直接將本機檔案拖曳到瀏覽器視窗即可上傳
- **多選操作**：支援 Ctrl+點擊 多選，Shift+點擊 範圍選擇
- **快捷鍵支援**：常用操作都有對應的快捷鍵

### 快捷鍵

| 快捷鍵      | 功能   |
| -------- | ---- |
| `Ctrl+C` | 複製   |
| `Ctrl+X` | 剪下   |
| `Ctrl+V` | 貼上   |
| `Delete` | 刪除   |
| `F2`     | 重新命名 |
| `Ctrl+A` | 全選   |

## 導覽

### 路徑導覽

頂部顯示當前路徑的麵包屑導覽， 點擊可以快速跳轉到任意上級目錄。

### 快捷按鈕

- **後退**：返回上一個存取的目錄
- **前進**：前進到下一個目錄
- **上級目錄**：返回上級目錄
- **重新整理**：重新整理當前目錄
- **主目錄**：返回預設目錄

## 檔案列表

檔案列表顯示以下資訊：

| 欄      | 說明                 |
| ------ | ------------------ |
| 名稱     | 檔案或目錄名稱            |
| 大小     | 檔案大小，目錄顯示「計算」可點擊計算 |
| 權限     | 檔案權限（如 0755）       |
| 所有者/群組 | 檔案的所有者和所屬群組        |
| 修改時間   | 最後修改時間             |
| 操作     | 開啟、壓縮、重新命名、刪除、更多   |

### 選擇檔案

![選擇檔案](/images/file/file-select.png)

選擇檔案後，頂部會顯示批次操作按鈕：

- **複製**：複製選中的檔案
- **移動**：移動選中的檔案
- **壓縮**：壓縮選中的檔案
- **權限**：修改權限
- **刪除**：刪除選中的檔案

### 更多操作

點擊檔案列的 **更多** 按鈕，顯示更多操作選項：

![更多操作](/images/file/file-context-menu.png)

- **複製**：複製檔案到其他目錄
- **移動**：移動檔案到其他目錄
- **權限**：修改檔案權限和所有者
- **壓縮**：壓縮檔案
- **複製路徑**：複製檔案的完整路徑
- **屬性**：檢視檔案詳細屬性

## 工具列

### 新建

點擊 **新建** 按鈕可以：

- 新建檔案
- 新建目錄

### 上傳

點擊 **上傳** 按鈕上傳本機檔案到伺服器。

支援的上傳方式：

- 點擊選擇檔案上傳
- **拖曳上傳**：直接將檔案拖曳到頁面即可上傳

### 遠端下載

點擊 **遠端下載** 按鈕，輸入 URL 位址，將遠端檔案下載到當前目錄。

適用場景：

- 下載軟體安裝套件
- 下載遠端備份檔案
- 從其他伺服器取得檔案

### 搜尋

在搜尋框輸入關鍵字搜尋檔案：

- 預設搜尋當前目錄
- 勾選 **包括子目錄** 可以遞迴搜尋

### 終端機

點擊 **終端機** 按鈕可以在當前目錄開啟終端機，方便執行命令列操作。

### 排序

點擊 **排序** 按鈕可以按不同欄位排序檔案列表。

## 文件編輯器

AcePanel 內建了功能強大的程式碼編輯器，基於 Monaco Editor（VS Code 同款編輯器核心）。

![文件編輯器](/images/file/file-editor.png)

### 編輯器特性

- **語法高亮**：支援各種程式語言的語法高亮
- **程式碼摺疊**：支援程式碼區塊摺疊
- **行號顯示**：顯示行號，方便定位
- **小地圖**：右側顯示程式碼縮圖
- **自動換行**：可切換自動換行模式
- **多檔案編輯**：支援同時開啟多個檔案，分頁切換

### 編輯器快捷鍵

| 快捷鍵            | 功能     |
| -------------- | ------ |
| `Ctrl+S`       | 儲存當前檔案 |
| `Ctrl+Shift+S` | 儲存所有檔案 |
| `Ctrl+F`       | 搜尋     |
| `Ctrl+H`       | 替換     |
| `Ctrl+G`       | 跳轉到指定行 |

### 編輯器工具列

- **儲存**：儲存當前檔案
- **儲存所有**：儲存所有修改的檔案
- **重新整理**：重新載入檔案內容
- **搜尋**：開啟搜尋面板
- **替換**：開啟替換面板
- **跳轉**：跳轉到指定行
- **字型大小**：調整編輯器字型大小
- **切換自動換行**：開啟/關閉自動換行
- **切換小地圖**：顯示/隱藏右側小地圖
- **設定**：編輯器設定

### 全螢幕模式

點擊 **最大化** 按鈕可以全螢幕編輯，獲得更大的編輯空間：

![全螢幕編輯器](/images/file/file-editor-fullscreen.png)

### 側邊欄檔案樹

編輯器左側顯示當前目錄的檔案樹，可以：

- 快速切換編輯其他檔案
- 新建檔案
- 搜尋檔案

### 狀態列

編輯器底部狀態列顯示：

- 檔案完整路徑
- 換行符類型（LF/CRLF）
- 游標位置（行、列）
- 縮排設定（空格/Tab）
- 檔案語言類型

## 權限說明

Linux 檔案權限由三組數字表示：

| 權限 | 數字 | 說明 |
| -- | -- | -- |
| r  | 4  | 讀取 |
| w  | 2  | 寫入 |
| x  | 1  | 執行 |

例如 `0755`：

- 所有者：7 (4+2+1) = 讀+寫+執行
- 所屬群組：5 (4+1) = 讀+執行
- 其他使用者：5 (4+1) = 讀+執行

常用權限：

- `0644`：普通檔案
- `0755`：可執行檔案和目錄
- `0600`：私密檔案（如金鑰）
