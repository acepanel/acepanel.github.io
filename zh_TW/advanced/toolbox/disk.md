# 磁碟

磁碟頁面提供磁碟分割管理、LVM 邏輯磁碟區管理、SMART 磁碟健康狀態以及 RAID 陣列狀態等功能。

## 磁碟管理

![磁碟管理](/images/toolbox/toolbox-disk.png)

### 磁碟資訊

頁面上方會顯示每顆磁碟的基本資訊：

- **磁碟名稱**：例如 vda、sda
- **磁碟類型**：SSD，或以大寫顯示的磁碟型號
- **容量**：磁碟總容量
- **分割區數**：分割區數量

承載根（`/`）分割區的磁碟會標示 **系統磁碟** 標籤，無法初始化或卸載其分割區。

### 分割區清單

每顆磁碟都會顯示其分割區資訊：

| 欄位    | 說明                   |
| ----- | -------------------- |
| 分割區名稱 | 分割區裝置名稱，例如 vda1、vda2 |
| 容量    | 分割區容量                |
| 已用    | 已用空間                 |
| 可用    | 可用空間                 |
| 使用率   | 使用百分比                |
| 掛載點   | 掛載目錄，例如 `/`、`/data`  |
| 檔案系統  | 檔案系統類型，例如 ext4、xfs   |
| 操作    | 卸載等操作                |

### 掛載分割區

將未掛載的分割區掛載到指定目錄：

- **分割區**：選擇要掛載的分割區
- **掛載路徑**：掛載目錄，例如 `/mnt/data`
- **掛載選項**：掛載參數，例如 `defaults,noatime`
- **開機時自動掛載**：是否寫入 fstab 以便開機時自動掛載

### 格式化分割區

:::danger 警告
格式化將清除分割區上的所有資料！
:::

- **分割區**：選擇要格式化的分割區
- **檔案系統類型**：ext4、ext3、xfs 或 btrfs

### 初始化磁碟

:::danger 警告
初始化將刪除磁碟上的所有分割區與資料！
:::

將整顆磁碟初始化為單一分割區：

- **磁碟**：選擇要初始化的磁碟
- **檔案系統類型**：ext4、ext3、xfs 或 btrfs

### 自動掛載設定（fstab）

顯示 `/etc/fstab` 中的掛載設定：

- **裝置**：裝置名稱或 UUID
- **掛載點**：掛載目錄
- **檔案系統**：檔案系統類型
- **選項**：掛載選項
- **操作**：移除設定

## LVM 管理

![LVM 管理](/images/toolbox/toolbox-disk-lvm.png)

LVM（邏輯磁碟區管理員）提供彈性的磁碟空間管理，並支援動態調整分割區大小。

### 實體磁碟區（PV）

實體磁碟區是 LVM 的基礎，通常是一個磁碟分割區或整顆磁碟。

**建立實體磁碟區**：

1. 選擇裝置（未使用的分割區或磁碟）
2. 點選 **建立實體磁碟區**

### 磁碟區群組（VG）

磁碟區群組由一個或多個實體磁碟區組成，相當於一個儲存池的概念。

**建立磁碟區群組**：

1. 輸入磁碟區群組名稱
2. 選擇要加入的實體磁碟區
3. 點選 **建立磁碟區群組**

### 邏輯磁碟區（LV）

邏輯磁碟區從磁碟區群組中配置空間，相當於傳統的分割區。

**建立邏輯磁碟區**：

1. 輸入邏輯磁碟區名稱
2. 選擇磁碟區群組
3. 設定大小（GB）
4. 點選 **建立邏輯磁碟區**

### 擴充邏輯磁碟區

動態擴充邏輯磁碟區的大小：

1. 選擇要擴充的邏輯磁碟區
2. 輸入擴充大小（GB）
3. 勾選 **自動調整檔案系統大小**（建議）
4. 點選 **擴充邏輯磁碟區**

:::tip 提示
LVM 的優勢在於邏輯磁碟區可以線上擴充，無需卸載分割區或重新啟動系統。
:::

## SMART

SMART 分頁會顯示支援 S.M.A.R.T. 的磁碟的健康資訊。

:::tip 提示
SMART 依賴 `smartctl`。 若尚未安裝，分頁會提示你先安裝 smartmontools（例如 `apt install smartmontools` 或 `dnf install smartmontools`）。
:::

選擇一顆磁碟即可在兩個分頁中檢視其詳細資訊：

- **基本資訊**：目前溫度、健康狀態（PASSED / FAILED），以及型號、序號、韌體、容量、介面、轉速與通電時數等裝置詳細資訊。
- **SMART 屬性**：完整的屬性表。 對於 ATA/SATA 磁碟，會列出 ID、屬性、值、最差值、臨界值、原始值與狀態；對於 NVMe 磁碟，會列出健康資訊記錄（例如已用百分比、可用備援空間、讀取/寫入資料單元數以及媒體錯誤）。

## RAID

RAID 分頁會自動偵測並顯示 RAID 陣列的狀態。 支援以下控制器類型：

- **Linux 軟體 RAID（mdadm）**
- **MegaRAID（LSI/Broadcom）**，透過 `storcli`
- **HP Smart Array**，透過 `ssacli`
- **Adaptec**，透過 `arcconf`

對於每個陣列，會顯示 RAID 等級、大小、等量大小、狀態、作用中/總裝置數以及重建進度（重建時），並附上實體磁碟清單。 對於硬體控制器，還會顯示控制器型號、序號、韌體與快取大小。

:::tip 提示
偵測硬體 RAID 需要安裝對應廠商的 CLI 工具。 若未偵測到 RAID 設定，分頁會提示你。
:::
