# 磁碟

磁碟頁面提供磁碟分區管理和 LVM 邏輯卷管理功能。

## 磁碟管理

![磁碟管理](/images/toolbox/toolbox-disk.png)

### 磁碟資訊

頁面頂部顯示每個磁碟的基本資訊：

- **磁碟名**：如 vda、sda
- **類型**：系統盤或資料盤
- **大小**：磁碟總容量
- **分區數**：分區數量
- **磁碟類型**：HDD、SSD 等

### 分區列表

每個磁碟下顯示其分區資訊：

| 列    | 說明                 |
| ---- | ------------------ |
| 分區名  | 分區裝置名，如 vda1、vda2  |
| 大小   | 分區容量               |
| 已用   | 已使用空間              |
| 可用   | 可用空間               |
| 使用率  | 使用百分比              |
| 掛載點  | 掛載目錄，如 `/`、`/data` |
| 檔案系統 | 檔案系統類型，如 ext4、xfs  |
| 操作   | 卸載等操作              |

### 掛載分區

將未掛載的分區掛載到指定目錄：

- **分區**：選擇要掛載的分區
- **掛載路徑**：掛載目錄，如 `/mnt/data`
- **掛載選項**：掛載參數，如 `defaults,noatime`
- **啟動時自動掛載**：是否寫入 fstab 開機自動掛載

### 格式化分區

:::danger 警告
格式化將清除分區上的所有資料！
:::

- **分區**：選擇要格式化的分區
- **檔案系統類型**：ext4、xfs、btrfs 等

### 初始化磁碟

:::danger 警告
初始化將刪除磁碟上的所有分區和資料！
:::

將整個磁碟初始化為單個分區：

- **磁碟**：選擇要初始化的磁碟
- **檔案系統類型**：ext4、xfs、btrfs 等

### 自動掛載配置（fstab）

顯示 `/etc/fstab` 中的掛載配置：

- **裝置**：裝置名或 UUID
- **掛載點**：掛載目錄
- **檔案系統**：檔案系統類型
- **選項**：掛載選項
- **操作**：移除配置

## LVM 管理

![LVM 管理](/images/toolbox/toolbox-disk-lvm.png)

LVM（Logical Volume Manager）提供靈活的磁碟空間管理，支援動態調整分區大小。

### 物理卷（PV）

物理卷是 LVM 的基礎，通常是一個磁碟分區或整個磁碟。

**建立物理卷**：

1. 選擇裝置（未使用的分區或磁碟）
2. 點擊 **建立物理卷**

### 卷組（VG）

卷組由一個或多個物理卷組成，是儲存池的概念。

**建立卷組**：

1. 輸入卷組名
2. 選擇要加入的物理卷
3. 點擊 **建立卷組**

### 邏輯卷（LV）

邏輯卷從卷組中分配空間，相當於傳統的分區。

**建立邏輯卷**：

1. 輸入邏輯卷名
2. 選擇卷組
3. 設定大小（GB）
4. 點擊 **建立邏輯卷**

### 擴展邏輯卷

動態擴展邏輯卷的大小：

1. 選擇要擴展的邏輯卷
2. 輸入擴展大小（GB）
3. 勾選 **自動調整檔案系統大小**（推薦）
4. 點擊 **擴展邏輯卷**

:::tip 提示
LVM 的優勢在於可以線上擴展邏輯卷，無需卸載分區或重新啟動系統。
:::
