# 終端機

終端機模組提供了基於 Web 的 SSH 終端機，可以直接在瀏覽器中連線伺服器執行命令。

## 終端機頁面

![終端機頁面](/images/ssh/ssh.png)

## 建立主機

點選 **建立主機** 按鈕新增 SSH 連線：

### 設定項目

- **名稱**：用於識別的連線名稱（留空時預設使用主機位址）
- **主機**：伺服器 IP 位址或網域名稱
- **連接埠**：SSH 連接埠，預設 22
- **驗證方式**：密碼或私密金鑰
- **使用者名稱**：SSH 使用者名稱
- **密碼**：登入密碼（僅在使用密碼驗證時顯示）
- **私密金鑰**：SSH 私密金鑰內容（僅在使用私密金鑰驗證時顯示）
- **金鑰密碼**：保護私密金鑰的密碼，若金鑰沒有密碼則留空（僅在使用私密金鑰驗證時顯示）
- **備註**：選填備註

### 驗證方式

| 方式   | 說明              |
| ---- | --------------- |
| 密碼   | 使用使用者名稱與密碼進行驗證  |
| 私密金鑰 | 使用 SSH 私密金鑰進行驗證 |

:::tip 建議
正式環境建議使用金鑰驗證，安全性較高。
:::

## 連線伺服器

1. 點選右上角工具列中的伺服器圖示，開啟 **主機** 抽屜
2. 點選主機卡片即可為其開啟一個終端機分頁
3. 或者點選分頁列中的 **+** 按鈕，從下拉選單中挑選一台主機
4. 開始執行命令

每個已連線的分頁都會顯示狀態指示器與目前的連線延遲（單位：毫秒）。

## 終端機功能

### 多重分頁

支援同時開啟多個終端機分頁，方便在多個工作階段間切換。

### 複製與貼上

- **複製**：選取文字時會自動複製，或使用 `Ctrl+Shift+C`（macOS 上為 `Cmd+C`）
- **貼上**：按右鍵即可貼上，或使用 `Ctrl+Shift+V`（macOS 上為 `Cmd+V`）

:::tip
在非 HTTPS 環境下無法使用剪貼簿 API，請改用 `Ctrl+V` 進行貼上。
:::

### 全螢幕模式

點選全螢幕按鈕即可進入全螢幕模式，獲得更大的終端機空間。

### 字型設定

你可以在設定彈出視窗中調整終端機字型大小（10–32），或在終端機上按住 `Ctrl` 並捲動滑鼠滾輪來縮放。

## 本機終端機

預設情況下，**本機** 分頁會在目前的伺服器上開啟一個 Bash 工作階段，讓你無需設定主機即可直接操作。

## 遠端主機

可以新增多個遠端主機，方便管理多台伺服器：

1. 開啟 **主機** 抽屜並點選 **建立主機**
2. 填寫遠端伺服器資訊
3. 儲存後，點選主機卡片即可連線

## 安全性建議

1. 使用高強度密碼或金鑰驗證
2. 修改預設 SSH 連接埠
3. 限制可存取 SSH 的 IP 位址
4. 定期更換密碼或金鑰
5. 使用非 root 使用者登入

## 常見問題

### 連線逾時

- 檢查伺服器 IP 與連接埠是否正確
- 檢查防火牆是否允許 SSH 連線
- 檢查 SSH 服務是否正常執行

### 驗證失敗

- 檢查使用者名稱與密碼是否正確
- 檢查金鑰格式是否正確
- 檢查使用者是否具有 SSH 登入權限

### 中文亂碼

- 檢查伺服器的 locale 設定
- 確保伺服器編碼為 UTF-8
