# 終端機

終端機模組提供了基於 Web 的 SSH 終端機，可以直接在瀏覽器中連接伺服器執行命令。

## 終端機頁面

![終端機頁面](/images/ssh/ssh.png)

## 創建主機

點擊 **創建主機** 按鈕添加 SSH 連接：

### 配置項

- **名稱**：連接名稱，用於標識
- **主機**：伺服器 IP 地址或域名
- **端口**：SSH 端口，默認 22
- **認證方式**：密碼或密鑰
- **用戶名**：SSH 用戶名
- **密碼/密鑰**：認證憑證
- **備註**：可選備註

### 驗證方式

| 方式 | 說明          |
| -- | ----------- |
| 密碼 | 使用用戶名和密碼認證  |
| 密鑰 | 使用 SSH 私鑰認證 |

:::tip 推薦
生產環境建議使用密鑰認證，更加安全。
:::

## 連接伺服器

1. 在左側主機列表中選擇要連接的主機
2. 點擊主機名稱打開終端機
3. 開始執行命令

## 終端機功能

### 多標籤

支持同時打開多個終端機標籤，方便在多個會話間切換。

### 複製貼上

- **複製**：選中文本後自動複製，或使用 `Ctrl+Shift+C`
- **貼上**：右鍵貼上，或使用 `Ctrl+Shift+V`

### 全屏模式

點擊全屏按鈕可以進入全屏模式，獲得更大的終端機空間。

### 字體設置

可以調整終端機的字體大小和字體類型。

## 本地終端機

默認會顯示本地伺服器的終端機連接，可以直接連接到當前伺服器。

## 遠程主機

可以添加多個遠程主機，方便管理多台伺服器：

1. 點擊 **創建主機**
2. 填寫遠程伺服器信息
3. 保存後在列表中選擇連接

## 安全建議

1. 使用強密碼或密鑰認證
2. 修改默認 SSH 端口
3. 限制 SSH 訪問的 IP 地址
4. 定期更換密碼或密鑰
5. 使用非 root 用戶登錄

## 常見問題

### 連接超時

- 檢查伺服器 IP 和端口是否正確
- 檢查防火牆是否允許 SSH 連接
- 檢查 SSH 服務是否正常運行

### 認證失敗

- 檢查用戶名和密碼是否正確
- 檢查密鑰格式是否正確
- 檢查用戶是否有 SSH 登錄權限

### 中文亂碼

- 檢查伺服器的 locale 設置
- 確保伺服器編碼為 UTF-8
